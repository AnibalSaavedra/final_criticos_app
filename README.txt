# Registro de Notificación de Valores Críticos

Aplicación web desarrollada en Streamlit para registrar y visualizar la notificación de valores críticos en laboratorio clínico.

## Valores críticos definidos:
- INR ≥ 3.5%
- Hematocrito ≤ 16%
- Glucosa ≤ 40 mg/dL
- Potasio ≥ 6.2 mEq/L
- Troponina ≥ 0.12 ng/mL

## Uso
Registra manualmente los datos solicitados y la aplicación calculará el tiempo de respuesta, clasificando automáticamente si la comunicación fue efectiva.
